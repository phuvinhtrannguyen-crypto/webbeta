# Cinematic video assets

Upload the two MP4 files here with these exact names:

- `all-in-left.mp4` - plays on the left when a player goes all-in
- `this-is-the-end.mp4` - plays on the right when a hand ends, at 2x speed

Vite serves files in `public/` from the site root, so the app loads them as:

- `/fx/all-in-left.mp4`
- `/fx/this-is-the-end.mp4`
